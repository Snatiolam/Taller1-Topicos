Dependencies:
    - Laravel v8
    - A mysql database named taller1_topicos
      (You can change the name of the DB modifying the .env file of the project)

      DB_HOST=127.0.0.1
      DB_PORT=3306
      DB_CONNECTION=mysql
      DB_DATABASE=taller1_topicos   // Your DB name
      DB_USERNAME=root              // Your DB username
      DB_PASSWORD=                  // Your DB password


Note:
    The username and the email are unique. If your repeat one of these fields an insertion error would occur.
    The code is not foolproof yet.

How to run the application
==========================

1. Start xampp or the database service (I used xampp):

    sudo /opt/lampp/lampp start

2. Once the database is running migrate the database so the tables can be created:

    php artisan migrate

This will create all the necessary tables so the project can run well.

If you want some random users to test the application you can run:

    php artisan db:seed

This will create 10 new random users.

3. Run the application (Recommended php artisan)

----------- PHP ARTISAN ----------------

- Starting the web service with php artisan can be done with the following command:

    php artisan serve

- Go to the navigation bar on your browser and type:
    
    http://127.0.0.1:8000

----------- XAMPP ---------------

If you prefer xampp you could run the app by:

    - Moving the project to /opt/lampp/htdocs and assigning user permissions to the project folder and the storage file.

    - Start xampp

    - Go to the navigation bar on your browser and type:

        localhost/Taller1_topicos/public
